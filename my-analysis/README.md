# RMDB 数据库管理系统 — 学习教程

本教程是一套面向**零基础**读者的数据库管理系统（DBMS）学习指南，以 RMDB（Relational Model Database）开源项目为参考，带你从零开始学会一个 DBMS 的**设计**与**实现**。

## 适合谁读

- 对数据库管理系统内核实现感兴趣的开发者
- 正在学习数据库系统概念、需要代码参考的学生
- 参加数据库系统竞赛（如全国大学生计算机系统能力大赛）的选手

## 阅读前提

- 了解 C 语言的基本语法（指针、结构体）
- 了解 C++ 的基本语法（类、智能指针、STL 容器）
- 其他所有数据库相关知识，本教程都会从头讲解

## 如何使用本教程

1. **按顺序阅读**：每章建立在前一章的基础上，不要跳读
2. **配合源码**：教程中标注了源码文件和行号（如 `rmdb.cpp:256`），建议打开源码对照阅读
3. **动手实践**：每学完一个章节，尝试修改代码、加日志、写测试，验证你的理解

## 约定

- 源码路径均相对于 `src/` 目录
- `文件名:行号` 表示指定文件的第几行代码
- 缩写术语在首次出现时给出全称
