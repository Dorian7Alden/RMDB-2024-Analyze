当在讲解文档中对某个概念连续追问了多个问题，说明这个概念重要且有价值，应该单独提取成一个独立文档进行讲解，然后在原文档中引用。不要让补充答疑的篇幅盖过了主要正文内容。

**Why:** 多轮追问的内容本身就是一个小专题，值得独立成文。挤在原文档里会喧宾夺主。

**How to apply:**
- 同一个概念被追问 2 次以上 → 考虑独立成文档
- 新文档放在同目录下，命名如 `05a-xxx.md` 插在相关章节之间
- 原文档中只保留一句话简要说明 + 指向新文档的链接
- 新文档收录所有相关答疑内容
