# Mermaid 画图规则

## 核心原则

**灵活运用，看得懂即可**，不局限于标准流程图形式。可以自由使用颜色标记不同类型信息块、添加备注文字说明。

## 规则列表

1. **禁用特殊字符**：mermaid 图中严禁使用 `()` `<>` `*` `/` 等特殊字符，类型信息用文字板书展示
2. **图例分离**：需要颜色/符号说明时，图例和主图分成两个独立图展示（先主图，后图例）
3. **子图嵌套表示从属关系**：对象间包含/层级关系用 subgraph 子图嵌套呈现
4. **层级子图用不同颜色**：subgraph 嵌套时每层用不同颜色区分，外层用冷色调，内层用暖色调，层次越深颜色越突出
5. **便利贴式备注块**：用特殊样式节点（虚线边框、特殊颜色）+ 虚线箭头 `-.->` 指向目标元素，充当便利贴备注
6. **不是所有内容都适合 mermaid**：对比矩阵、条件判断、多维度比较用表格更直观，流程图、层级关系、时序才用 mermaid

## 来源

合并自 memory 目录下 5 个 mermaid 相关规则文件。涉及 mermaid 图表时读取本文件。
