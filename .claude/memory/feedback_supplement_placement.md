补充说明、答疑、锁解释等辅助内容应放在正文主要内容的后面，而不是前面。

**Why:** 读者先看到正文产生疑问，往下读时自然找到解答。放在前面会打断阅读节奏，读者还不知道正文是什么就先看到一堆解释。

**How to apply:**
- 前置知识（读者必须先知道才能理解正文的）→ 放前面
- 补充说明、答疑、语法解释（对正文的额外说明）→ 放后面
- 判断标准：没这段说明能不能先看正文？能 → 放后面；不能 → 放前面
