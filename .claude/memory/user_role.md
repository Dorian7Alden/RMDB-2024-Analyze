用户是数据库管理系统（DBMS）的初学者，没有相关基础。

## 目标
- 通过 RMDB（Rucbase）参考项目学会数据库管理系统的**设计**与**开发**
- 学习的最终产出是一套完整的教程文档（放在 `my-analysis/` 目录下），能讲给别人听、让别人也跟着学会
- 我（Claude）的角色是指导老师，负责答疑解惑、帮助整理文档

## 教程文档要求
- 只修改 `my-analysis/` 目录下的文档，源代码一行不改
- 每份教程文档尽可能小，拆分文件、分包，分小章节，一次只看一小部分
- 结合 mermaid 画图 + 源代码解析，给出具体实现案例，对应到源码
- 不适合 mermaid 的用文本块+纯文字板书（如项目结构用文字版文件树）
- 先讲顶层、抽象的组成部分，先说清楚是什么东西，再讲具体实现
- 缩写、变量名等要给出全称说明
- 假设读者是零基础小白，没有前置知识，要补充必要的前置概念
- 教程应保持随时能出版的状态，风格统一，排版统一
- 发现有内容对不上的要及时调整
- 文档撰写有先后顺序，按知识体系推进：先基础后复杂

## 项目背景
- RMDB 是中国人民大学数据库教学团队开发的 DBMS 框架
- 用于全国大学生计算机系统能力大赛数据库管理系统赛道
- 语言：C++17，构建：CMake
- 开源协议：木兰宽松许可证 v2
