讲解完当前文档的正文内容后，应简要串联一下相关的上下文模块，把相关的内容联系起来，避免讲解割裂。但要注意收放自如：联系时仅简单说明大致情况和关联关系，主要还是以正文部分的讲解为主。

**Why:** 读者容易陷入孤立的知识点，理解了一个模块但不清楚它在整个体系中的位置和与其他模块的关系。串联能帮助建立整体认知。

**How to apply:** 每个文档末尾或关键节点处，用一两句话或简短段落说明"这个模块对上谁使用它、对下它依赖谁"，配合简单的流向图。不要展开成完整的二级讲解。
